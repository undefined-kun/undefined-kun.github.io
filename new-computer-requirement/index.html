新电脑之所需

新电脑之所需

考虑到寒假之后粉丝群直播的需要，新电脑将是 Windows 系统而非 macOS，故将此文用于预备，并造福他人。

软件与工具安装的整体指导：

  • 分区仅分 C 盘（系统盘）和 D 盘（数据盘），一切软件安装在 D 盘；各种文件夹建立于 D 盘，且须科学合理。
  • 以管理员权限运行，并为所有用户安装软件。

浏览器：Chrome、Tor

Google Chrome: 官方下载地址。

Tor Browser: 官方下载地址。

Tampermonkey 扩展脚本

Pixiv 增强：https://greasyfork.org/zh-CN/scripts/34153-pixiv-plus。

高匿代理：Geph

Geph: 官方免翻墙镜像。

虚拟化软件：VMware

顾及 VirtualBox 并不完全开源、不支持快照功能和易用性等问题，笔者最终选择拥抱 VMware。

VMware Workstation Pro: 官方下载地址。

文件搜索工具：Everything、Listary

Everything: 官方下载地址。

Listary: 官方下载地址。

文件归档工具：7-Zip

笔者并不需要 Bandizip 的智能解压缩功能，加之 7-Zip 简洁高效且开源而又实用，我宣布 7-Zip 天下第一，不接受反驳。

7-Zip: 官方下载地址。

磁盘空间分析工具：SpaceSniffer

SpaceSniffer: 官方下载地址。

硬件详情查看工具：CPU-Z、GPU-Z

CPU 和 GPU 显然是最重要的两大硬件，因此，查看其具体信息，以高效地解决利用它们时所遇到的疑难，是必要的。

CPU-Z: 官方下载地址。

GPU-Z: 官方下载地址。

终端：Windows Terminal + Oh My Posh

Windows Terminal: 直接从应用商店安装即可。

Oh My Posh: 官网。

版本控制软件：Git

Git for Windows: 官网。

文本编辑器：VSCode、Sublime

由于太过强大，笔者将 VSCode 用作工程用途；而 Sublime 则专为刷算法题而使用。

Visual Studio Code: 官方下载地址。

Sublime Text: 官方下载地址。

LaTeX 发行版：TeX Live

可参阅 这篇、这篇 和 这篇 文章。

音乐创作软件：FL、SoVITS、袅袅

以前笔者浅玩过 VOCALOID，然而上手容易调教难，现在还是决定改做 AI 推演了，反正又不缺音源；袅袅仅用于填词。

FL Studio: 官方下载地址。

SoVITS: 官方下载地址。

袅袅虚拟歌手：官方下载地址。

音乐播放软件：网易云音乐

显然要记得取消勾选创建桌面快捷方式、开机启动、添加到任务栏、设为默认音乐播放器这几项野蛮的东西。

网易云音乐：官方下载地址。

图像编辑软件：GIMP、Scratch

GIMP: 官方下载地址。

Scratch: 官方下载地址。

录屏与串流软件：OBS、直播姬

OBS Studio: 官方下载地址。

哔哩哔哩直播姬：官方下载地址。

即时通讯软件：QQ、微信、钉钉、飞书、Telegram

钉钉和飞书为特殊时期预备；Telegram 可规避审查，虽则不常用，但似乎一些知名 OIer 都使用此软件，笔者故而效仿。

QQ: 官网。

微信：官网。

钉钉：官方下载地址。

飞书：官方下载地址。

Telegram: 官网。

游戏平台：Steam

事实上，Steam 上不止有游戏，更有 Wallpaper Engine。

Steam: 官方下载地址。

BitTorrent 客户端：qBittorrent、迅雷

qBittorrent: 官方下载地址。

迅雷：官网。

资源下载：百度网盘

显然，这玩意儿既不优雅，也不实用，但又不得不用。

百度网盘：官方下载地址。
